== Description ==

A Simple Wordpress Plugin That Disables All Wordpress Core Updates



= Please note!!! =


It is important to keep your WordPress Core up to date. 
If you use this plugin, make sure your wordpress core version is not harming your website.

== Installation ==


1. Go to the Plugins> Add New> Upload in you admin dashboard.

2. Upload the zip file and install.
3. Go to the Plugins section in your admin dashboard and activate.
